Everything runs on your machine

No API keys. No subscriptions. No data leaving your laptop.

🔒

100% Local & Private

Your data never leaves your machine. No cloud calls, no telemetry, no tracking. What happens in Folk stays in Folk.

📁

File System Agent

Folk can read, write, and organize files in your workspace. Scoped by default, with dialog-gated access to files outside.

🔄

Multi-Step Planning

Powered by Claude Agent SDK. Folk plans, executes tools, and iterates — handling complex tasks that take multiple steps.

🔌

MCP Integration

Connect to external services via Model Context Protocol. Slack, Gmail, databases, and any custom MCP server.

⚙

Swap Models Anytime

Ships with Gemma 4 E4B (4.5B params). Drop in any GGUF model, or point to Ollama for bigger models.

💻

Cross-Platform

macOS (Apple Silicon & Intel), Windows, and Linux. Native GPU acceleration via Metal, CUDA, and Vulkan.
