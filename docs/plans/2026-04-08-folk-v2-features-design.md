# Folk v2 Features Design

**Date:** 2026-04-08
**Status:** Approved

## Features

### 1. Keyboard Shortcuts
Global shortcuts via Electron accelerators + renderer key handlers.

### 2. File Drag & Drop
Drop zone in chat input area with visual feedback overlay.

### 3. Toast Notifications
Lightweight notification system for errors, confirmations, warnings.

### 4. Conversation Export
Export to Markdown via context menu or keyboard shortcut.

### 5. Window State Persistence
Save/restore window bounds, sidebar/panel visibility across restarts.

### 6. Context Window Indicator
Token count display, context usage warning, auto-summarize.

### 7. Inline Conversation Rename
Double-click to rename in sidebar.

### 8. Stop Generation Button
Visible stop button during agent processing.
